# Digital Board Games

## Getting Started
* Install npm
* `sudo npm install -g firebase-tools`
* `sudo npm install firebase-bolt`
* `firebase login`
* `firebase serve`
* Now navigate to [localhost:5000](http://localhost:5000)

## Deploying

* `firebase deploy`
